# SmartWhereSDK

## Installation

SmartWhereSDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
  use_frameworks!

  pod 'SmartWhereSDK', '~> 16.354.0'
```
